## TRS-80 CoCo keyboard replacements
Here are some projects for keyboard replacements for the TRS-80 Colour Computer.

For more information on the project, please consult [the Leaded Solder blog post on the keyboard PCB](https://www.leadedsolder.com/2025/01/07/coco-mechanical-keyboard-build.html).

## Branches
### master
This branch contains a 3D printable plate for fly-wiring switches. It is not tested and not recommended for production.

### kicad-pcb
This branch contains a KiCad-developed PCB for mounting Gateron-style switches. It has been tested. You can [get the gerbers from the Releases page](https://github.com/barbeque/coco-plate-keyboard/releases).
